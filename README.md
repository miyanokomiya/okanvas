## commnad

``` bash

$ yarn install

# start dev & demo server
$ yarn demo

# lint
$ yarn lint

# test
$ yarn test [--watch]

# build
$ yarn build
```

## publish
Update `version` in `package.json`, commit with a comment `Release x.x.x` and merge into the `main` branch.
