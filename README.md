# okanvas

parse SVG to polygons

## usage

``` bash
yarn add okanvas
```

```js
import * as okanvas from okanvas
```

## commnad

``` bash
# install dependencies
$ yarn install

# lint
$ yarn lint

# test
$ yarn test [--watch]

# build
$ yarn build
```
